# Kubernetes and Cloud Native Security Associate (KCSA)

Welcome! This folder is a curated set of resources I used to successfully prepare for the **KCSA** certification. My goal is to help others navigate their own certification journey with practical, proven materials.

## üìñ About the Exam

The **KCSA** validates foundational knowledge of security concepts and technologies within the cloud-native ecosystem.

**Exam Details:**
- **Format:** Multiple Choice(one correct answer)  
- **Duration:** 90 minutes  
- **Passing Score:** 75%  

---

## üß© Exam Domains & Competencies

| Domain | Weight | Key Topics |
|------|--------|-----------|
| Cloud Native Security Overview | 14% | 4Cs of Cloud Native Security, cloud provider & infrastructure security, security controls & frameworks, isolation techniques, image & artifact security, workload and application code security |
| Kubernetes Cluster Component Security | 22% | API server, controller manager, scheduler, kubelet, container runtime, kube-proxy, pods, etcd, container networking, client security, storage |
| Kubernetes Security Fundamentals | 22% | Pod Security Standards & Admissions, authentication & authorization, secrets management, isolation & segmentation, audit logging, network policies |
| Kubernetes Threat Model | 16% | Trust boundaries & data flow, persistence, denial of service, malicious code execution, network attackers, access to sensitive data, privilege escalation |
| Platform Security | 16% | Supply chain security, image repositories, observability, service mesh, PKI, connectivity, admission control |
| Compliance & Security Frameworks | 10% | Compliance frameworks, threat modeling frameworks, supply chain compliance, security automation & tooling |
