# 🚀 Mission Kubestronaut

**Mission Kubestronaut** is my personal journey through the **Kubernetes and Cloud Native certifications** - from learning the fundamentals to mastering production-grade systems.  

This repository contains **resources and study materials** I’ve used to prepare for each exam on the path to becoming a certified *Kubestronaut* 👨‍🚀  

---

## 🌌 Mission Goals

- Build a strong foundation in **Cloud Native technologies**
- Complete the **Kubernetes certification path** (KCNA → CKA → CKAD → KCSA → CKS)
- Share practical study resources and tips for each milestone
- Help others navigating the same journey
